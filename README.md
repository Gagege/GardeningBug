# 🌱 GardeningBug

A thoughtfully designed Android app for plant care and garden management, specifically crafted for seniors. GardeningBug combines simplicity with powerful features to help you maintain your garden with confidence and ease.

## Features

- 🌿 **Simple Plant Management**: Easy-to-use interface for adding and organizing your plants
- 💧 **Clear Watering Reminders**: Never miss a watering with gentle, customizable reminders
- 📱 **Senior-Friendly Design**: Large text, high contrast, and intuitive navigation
- 🔍 **Easy Organization**: Simple categorization of plants (Vegetable, Herb, Flower, Fruit, Tree)
- 📝 **Simple Notes**: Keep track of important information with an easy-to-read format
- 🔄 **Straightforward Sorting**: Sort plants by name, type, watering needs, or planting date
